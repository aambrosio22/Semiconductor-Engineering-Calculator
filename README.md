# Semiconductor Engineering Calculator

## Overview

This project is a Python-based engineering calculator developed to explore fundamental electrical concepts and their connection with semiconductor devices.

The project starts from basic electrical equations and progressively evolves toward semiconductor material analysis and device characterization.

The main objective is to connect:

Physics concepts → Mathematical models → Python implementation → Engineering analysis

---

# Project Evolution

## Version 1  — Interactive Engineering Calculator

This version implements fundamental electrical calculations by introducing a more structured software approach. 

## Implemented Features

The calculator can perform:

### Power Calculation

P = V × I


### Voltage Calculation

V = R × I


### Resistance Calculation

R = V / I


### Energy Calculation

E = P × t

# Semiconductor Engineering Connection

Although the project is based on fundamental electrical equations, these concepts represent the foundation of semiconductor device analysis.

Electrical power is directly related to:

- transistor power dissipation;
- thermal management;
- device reliability;
- leakage current.

Resistance calculations are connected to:

- semiconductor material properties;
- silicon resistivity;
- doping concentration;
- device geometry.

Energy analysis is related to:

- power efficiency;
- low-power electronics;
- modern semiconductor scaling.
